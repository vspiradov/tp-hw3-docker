# Домашнее задание №3. Docker. Bash

## Как открыть HTML-отчёт (Задание для терминаторов)
По условию необходимо было использовать образ, который встречался на семинарах. Был выбран `python:3.12-slim`, с помощью которого поднимается встроенный HTTP-сервер, раздающий файлы из папки `/data`.

Шаги как запустить проект(в среде GitHub Codespaces):
1. Соберите и запустите генератор и аналитик:
   `./run.sh build_generator && ./run.sh run_generator`
   `./run.sh build_reporter && ./run.sh run_reporter`
2. Запустите веб-сервер командой: 
   `./run.sh report_server`
3. В нижней панели Codespaces перейдите во вкладку **Ports** (Порты).
4. Найдите порт **8080** (Local Address).
5. Нажмите правой кнопкой мыши по столбцу `Visibility` (Видимость) и измените значение в колонке видимость порта на **Public**.
6. Кликните по ссылке локального адреса порта 8080. В браузере откроется список файлов.
7. Нажмите на `report.html`, чтобы посмотреть сгенерированный отчет.
